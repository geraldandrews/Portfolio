About Me

Hello, my name is Gerald and I live in Bluefield, VA and I enjoy football, the outdoors and spending time with family. For the last eight years I have also enjoyed being a self taught content creator through designing and developing my own front-end web development and graphic design projects. My passion and goal for each project is to learn something new and create something that is user friendly, mobile responsive, stands out to people, and to learn and develop new skills which would be useful for companies or potential clients that could use my services.

Skills and Tools

HTML and CSS

Proficient with a solid understanding of HTML5 and CSS3. I have used both for years now to create the most user friendly and quality websites and apps for my own personal websites and projects.

Javascript

Javascript is the world's most popular programming language and is the language of the web. I like to use javascript to create the most interactive and dynamic websites and apps. I continue to use it and experiment with it daily to become a better developer.
